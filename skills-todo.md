# Skills to create

- Zettelkasten / research wiki
  - Help the agent create a Zettelkasten system to organize research on a
    topic. This could be used for an individual project or for a user's
    general knowledge management.
- Showboat
  - A skill that creates demonstrations of how to use a particular tool or
    library or a walkthrough of a particular process. Tools of use would be
    `showboat` and related, plus `asciiscinema` for recording terminal
    sessions.
- Spec-driven development
  - A skill that colaborates with the user to produce EARS and Gherkin
    specifications for a project, and then uses those specifications to drive
    the development of the project.
- Project scaffolding
  - Python, Typescript, Go, etc.
  - Renovate, Mergify, etc.
  - Pre-commit hooks, CI/CD pipelines, etc.
  - Best practices for project structure, documentation, testing, etc.
    - Package manager cooldown, dependency locking
- Renovate skill
  - A skill that helps users set up and configure Renovate for their projects,
    including creating configuration files, building custom rules, and
    validating the setup (and rules/special markers).
